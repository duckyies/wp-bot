Client

Client Entry point for the WhatsApp bot.

Initializes the WhatsApp client, connects to the MongoDB database, and handles QR code generation for authentication when the bot is ready.

Other things:

  • Loading command and event handlers when the bot is ready.
  • Handling uncaught exceptions and signals for graceful shutdown.
