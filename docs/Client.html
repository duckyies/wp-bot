Client

Source: index.js:6

Client Entry point for the WhatsApp bot.

Initializes the WhatsApp client, connects to the MongoDB database, handles QR code generation for authentication, and loads command and event handlers when the bot is ready.

Other things:

  • Loading command and event handlers when the bot is ready.
  • Handling uncaught exceptions and signals for graceful shutdown.
