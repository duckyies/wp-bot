# wp-bot

## Features
- UNO
- Poketwo bot
- Chat bots
- NYT Games wordle,crossword
- Chess puzzles

## Installation
```powershell
npm i -g yarn@latest
yarn install
```

## Starting the bot

### With pm2
```powershell
npm start
```

### Without pm2
```powershell
node dist/index.js
```

## Commands
All commands can be found within src/commands/
