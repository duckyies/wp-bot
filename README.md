# wp-bot

## Documentation
Documentation for all commands can be found [here](https://scaranaraa.github.io/wp-bot/)

## Features
- UNO
- Poketwo bot
- Chat bots
- NYT Games wordle,crossword
- Chess puzzles

## Installation
```bash
npm i -g yarn@latest
yarn install
```

## Starting the bot

### With pm2
```bash
npm start
```

### Without pm2
```bash
node dist/index.js
```

## Commands
All commands can be found within src/commands/
